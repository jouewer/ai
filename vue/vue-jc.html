安装教程

vue搭建，以及创建项目教程

网址：https://nodejs.org/en 检查：node -v 安装项目：npm init vue@latest 启动前: npm install 启动：npm run dev 安装插件：Vue Language Features(Volar) 解析：npm i

vue3启动 进入文件夹

一般在 vue 3.x 的时候用 npm run serve vue2.x项目运行的时候用到 npm run dev

创建项目 vue create my-project(项目名称)

或者

npm init vue@latest

vue.config.js文件添加lintOnSave:false/*关闭语法检查*/

在项目cmd，进行打开编程软件 code . 打开的是这个软件：Visual Studio Code

命令npm config get registry，查看当前npm镜像源，是taobao的，切换回默认的试试看。npm config delete registry

重新执行npm install，完美

1.打开终端，输入以下命令，设置淘宝镜像源： npm config set registry https://registry.npm.taobao.org

2. 输入以下命令，查看是否设置成功： npm config get registry （https://registry.npm.taobao.org/，说明配置的是淘宝镜像。）

解决Code 提示因为在此系统上禁止运行脚本。有关详细信息

1、以管理员身份运行VS Code

2、运行命令 get-ExecutionPolicy 如果显示Restricted表示状态是禁止的

3、运行命令 set-ExecutionPolicy RemoteSigned

4、再次运行命令验证 get-ExecutionPolicy 如果显示RemoteSigned表示状态是允许的

项目打包 cnpm run build 预览打包后的项目 npm run preview
